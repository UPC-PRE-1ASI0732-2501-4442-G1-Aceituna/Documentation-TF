<div id='3.2.'><h3>3.2 User Stories.</h3></div>

Las user stories permiten convertir el lenguaje cotidiano de los usuarios en requisitos de software que deben tenerse en cuenta durante el desarrollo del sistema. Cuando están bien formuladas, brindan al equipo de desarrollo una descripción clara sobre la funcionalidad que se va a implementar, su objetivo y el valor que aporta al usuario. En el caso del producto EcoMovil, el equipo EcoMovil Squad ha definido un conjunto de user stories enfocadas en la creación de la Landing Page, la aplicación web y aspectos técnicos específicos.
